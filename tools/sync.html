How It Works

1

You edit files

Use VS Code, Cursor, Vim ‚Äî any editor you want. weld-sync watches silently.

2

Auto-commit

After 5 seconds of quiet, changes are committed to jj (Jujutsu). Your work is always versioned.

3

Sync to team

Changes push to the sync server. Your teammates' daemons pull them down. Everyone stays in sync.

4

Auto-merge

jj automatically rebases changes. Non-overlapping edits merge clean. Conflicts are surfaced but don't block.

Quick Start

# Install
npm install -g weldr

# Initialize in your project
cd my-project
weldr init

# Start the daemon (watches files, auto-commits)
weldr daemon

# Connect to team sync server
weldr daemon --sync-url wss://your-server.com --sync-token your-token

# Check status anytime
weldr status
